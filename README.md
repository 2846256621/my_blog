# my_blog
个人博客
##### 1.项目功能：
- 个人博客的添加
- 展示
- 修改
- 删除
- 搜索
- 按年份归档 
- 关于个人
- 留言及留言展示
##### 2.项目使用的技术
- 使用了 HTML/CSS/JavaScript
- 使用了art_template模板渲染
- 后台使用node的express框架
- 连接了mysql的数据库
- 做了手机的响应式
##### 3.总结
- 项目很小，但是使用的技术比较多，实现了前端加后台。
- 系统的进行学以致用吧
